Meteor is Shopware's open-source design system for building consistent administrative product experiences across the Shopware ecosystem.

## Packages

Meteor is published as three independent npm packages. Use all three together or install only what you need.

| Package                                 | Description                                                         |
| --------------------------------------- | ------------------------------------------------------------------- |
| `@shopware-ag/meteor-component-library` | Vue 3 component set with built-in tokens, icons, and the Inter font |
| `@shopware-ag/meteor-tokens`            | Design tokens as CSS custom properties, works in any framework      |
| `@shopware-ag/meteor-icon-kit`          | SVG icon set, usable standalone or as Vue components                |

## Installation

Install the component library:

```bash
npm install @shopware-ag/meteor-component-library
```

Import the font and styles in your entry file:

```ts
import "@shopware-ag/meteor-component-library/font.css";
import "@shopware-ag/meteor-component-library/styles.css";
```

## Usage

Import components individually to keep your bundle lean:

```ts
import { MtButton } from "@shopware-ag/meteor-component-library";
```
