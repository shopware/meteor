## Why Meteor

Without a shared system, every team makes its own decisions about color, spacing, interaction, and language. The result is products that feel inconsistent, and a development process that wastes time re-solving problems that have already been solved elsewhere.

Meteor gives designers and developers a single source of truth. By aligning Figma components, implementation components, and design tokens, it closes the translation gap between design and engineering: decisions are made once, encoded in tokens and components, and applied consistently everywhere. This reduces misalignment, accelerates delivery, and lowers the risk of visual drift as products grow.

The payoff is compounding: teams ship faster because they are building on a stable foundation, products feel more polished because the same decisions apply everywhere, and design and engineering stay aligned because they are working from the same artifacts. Being open source means Meteor benefits from contributions across the Shopware partner and developer community. It is the foundation Shopware and its ecosystem build on together.

## What's in Meteor

Meteor is published as three independent npm packages. Use all three together or install only what you need.

::card-group
  :::card{icon="i-lucide-blocks" title="@shopware-ag/meteor-component-library" to="https://www.npmjs.com/package/@shopware-ag/meteor-component-library" target="_blank"}
  Vue 3 component set with built-in tokens, icons, and the Inter font.
  :::

  :::card{icon="i-lucide-palette" title="@shopware-ag/meteor-tokens" to="https://www.npmjs.com/package/@shopware-ag/meteor-tokens" target="_blank"}
  Design tokens as CSS custom properties, works in any framework.
  :::

  :::card{icon="i-lucide-shapes" title="@shopware-ag/meteor-icon-kit" to="https://www.npmjs.com/package/@shopware-ag/meteor-icon-kit" target="_blank"}
  SVG icon set, usable standalone or as Vue components via mt-icon.
  :::
::

## FAQ

::accordion
:::accordion-item{label="Is Meteor free to use?"}
Yes. Meteor is completely free and open source under the MIT license. You can use it for personal, commercial, or any other projects without restrictions.
:::

:::accordion-item{label="Which frameworks does Meteor support?"}
Meteor components are built specifically for Vue.js 3 and above. The design tokens and icon kit are framework-agnostic and can be used with any framework or with vanilla JavaScript and CSS.
:::

:::accordion-item{label="Which browsers does Meteor support?"}
Meteor supports all modern evergreen browsers, including the latest versions of Chrome, Firefox, Safari, and Edge.
:::

:::accordion-item{label="Do I need to credit Shopware?"}
Attribution is not required, but it is appreciated. You can link to [meteor.shopware.com](https://meteor.shopware.com) or mention that you are using Shopware's Meteor Design System.
:::

:::accordion-item{label="Where can I get support?"}
Open an issue on [GitHub](https://github.com/shopware/meteor/issues) for bugs and feature requests.
:::

::
